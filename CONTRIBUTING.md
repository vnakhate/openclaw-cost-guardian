# Contributing to OpenClaw Cost Guardian

Thanks for your interest in contributing!

## Getting Started

1. Fork the repo
2. Clone your fork
3. Install the skill locally:

```bash
mkdir -p ~/.openclaw/skills/cost-guardian
cp SKILL.md ~/.openclaw/skills/cost-guardian/SKILL.md
openclaw gateway restart
```

## Reporting Issues

- Use the **Bug Report** template when [opening a new issue](../../issues/new/choose)
- Use the **Feature Request** template for ideas

## Submitting Changes

1. Create a branch from `main`
2. Make your changes
3. Update `CHANGELOG.md` if applicable
4. Open a pull request
